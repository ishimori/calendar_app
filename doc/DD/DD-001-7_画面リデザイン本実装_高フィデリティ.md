## 目的

[DD-001-6/design/](DD-001-6/) の高フィデリティデザイン案（シート1=業務一覧 / シート2=ガント / 詳細編集モーダル）を、本プロジェクト（React + Vite + TypeScript + react-query）に再現実装する。配色・タイポグラフィ・余白・インタラクションをピクセル単位で再現する。

## 背景・課題

- DD-001-5 までのシート1/2 は、データモデル動作確認用のミニマル UI（テーブル直書き + 自作 Grid ガント）
- 業務で運用するには情報密度・操作性・視覚的グルーピング（タンク）が不足
- Claude Design でリファレンスを起こした → 同じものを実装に落とす
